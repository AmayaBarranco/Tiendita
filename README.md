# Tiendita La Abuelita
Proyecto estudiantil de una pagina web informativa para la Unidad 2.

## Descripción
Esta página web muestra la oferta de productos y comidas preparadas de la Tiendita La Abuelita. 
Incluye:
- Sección de bienvenida con imagen del negocio.
- Menú de productos y comidas preparadas.
- Listado de precios en tablas.
- Ubicación mediante mapa embebido. (Profe no supe poner un punto para la ubicacion exacta disculpe).
- Formulario de contacto funcional.
- Enlace a archivo XML con catálogo de productos. 

## Cómo ver el archivo XML
Haz clic en "Ver Catálogo XML" en el menú superior, o abre directamente `datos.xml` desde tu navegador.
